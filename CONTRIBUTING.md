# Contributing to Kreeda-Ankana

Thank you for your interest in contributing to **Kreeda-Ankana**! We welcome contributions from the community to make sports management easier for everyone.

## 🚀 How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issues tracker.
- Describe the bug and provide steps to reproduce it.
- Include information about your device and Android version.

### 2. Suggesting Enhancements
- Open an issue with the tag `enhancement`.
- Explain why the feature would be useful and how it should work.

### 3. Pull Requests
- Fork the repository.
- Create a new branch (`feature/your-feature-name` or `bugfix/your-bug-name`).
- Commit your changes with clear, descriptive messages.
- Ensure your code follows the existing style and is well-documented.
- Submit a pull request to the `main` branch.

## 🛠️ Development Setup
1. Follow the **Installation** steps in the `README.md`.
2. Use **Android Studio** for development.
3. Follow the **Kotlin Style Guide** for code formatting.
4. Ensure all new features are tested on an emulator or physical device.

## 📜 Code of Conduct
Please be respectful and professional in all interactions within this project.
